# Rock Paper Scissors Game

This is a simple and classic Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## How to Play

1.  Open the `index.html` file in your web browser.
2.  You will see three options: Rock, Paper, and Scissors.
3.  Click on your choice.
4.  The computer will randomly make its choice.
5.  The winner of the round will be displayed.
6.  To play again, simply choose another option.

## Technologies Used

*   HTML
*   CSS
*   JavaScript
